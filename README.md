# Nightclub Guestlist MVP

SvelteKit MVP for nightclub table reservations, guestlist RSVP, and door check-in.

## Stack
- SvelteKit + TypeScript
- Tailwind CSS
- shadcn-svelte style UI primitives in `src/lib/components/ui`
- Firebase Auth (phone OTP), Firestore, Hosting/Emulator config

## Start
```sh
npm install
npm run check
npm run dev
```
